# qianku-music
#### 支持h5与微信小程序运行
#### 本项目仅限学习交流使用, 不得用于商业用途, 如作他用所承受的法律责任一概与作者无关

## 安装

```shell
$ git clone https://github.com/Ye-wz/qianku_music.git
$ npm run install-all
```

## 运行服务层

```shell
npm run server
```

### 热编译
```
npm run dev:h5
npm run dev:mp-weixin
```
